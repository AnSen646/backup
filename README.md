备份虾米收藏歌曲，导出歌曲风格标签

## 功能
- [x] 备份收藏歌曲音频资源
- [x] 收藏歌曲标签可视化
- [x] 导出收藏歌曲原始数据
- [ ] 备份其他类型的收藏数据
