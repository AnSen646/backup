备份虾米收藏歌曲，导出歌曲风格标签

***目前还能正常工作！***

## 功能
- [x] 根据虾米ID一键备份收藏数据，支持专辑、歌曲、艺人、歌单(自建/收藏) 导出为JSON
- [x] 批量备份音频资源
- [x] 收藏歌曲风格标签的可视化
- [x] 备份数据查看可排序（如发现自己播放量最高的原创歌单

## 如何使用
> 输入你的虾米数字ID 
>
> https://www.xiami.com/user/xxxx
>
> xxxx链接尾部的数字 就是你的虾米ID
> 
> 备份文件夹用英文文件名 如: F:\allsong
